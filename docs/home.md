# a8n Landing Page Structure

This document outlines the structure and content for the a8n landing page. a8n is an AI-powered workflow automation platform designed to be a high-performance, developer-friendly alternative to a8n and Zapier.

## 1. Hero Section
- **Headline:** Build the Future of Automation with AI-Native Workflows.
- **Subheadline:** The ultimate platform to design, build, and execute intelligent automation pipelines. Visual, durable, and type-safe from the ground up.
- **Primary CTA:** Get Started for Free (links to `/signup`)
- **Secondary CTA:** View Documentation (links to `/docs`)
- **Visual:** A high-fidelity screenshot or video of the a8n visual editor in action.

## 2. Visual Workflow Editor (Core Feature)
- **Heading:** Visualize Your Logic.
- **Description:** Our drag-and-drop DAG editor built on React Flow gives you complete control over your automation. Snap-to-grid, minimap, and auto-layout make building complex workflows a breeze.
- **Key Points:**
    - Real-time status streaming.
    - Interactive node configuration.
    - Step-by-step execution feedback.

## 3. The Power of AI Nodes
- **Heading:** Intelligent Automation, Native to the Core.
- **Description:** Seamlessly integrate the world's most powerful AI models directly into your workflows.
- **Supported Models:** OpenAI, Anthropic, Google Gemini.
- **Use Cases:** 
    - Automated content generation.
    - Intelligent lead scoring.
    - AI-powered customer support routing.

## 4. Key Platform Features (Bento Grid)
- **Durable Execution:** Powered by Inngest. Workflows never fail silently. Automatic retries and failure recovery built-in.
- **10+ Native Nodes:** Triggers (Stripe, Google Forms, Manual) and Executors (HTTP, Discord, Slack, AI).
- **Encrypted Vault:** AES-256 encryption for your API keys and credentials via Cryptr.
- **Real-Time Monitoring:** Live audit trails of every workflow run with detailed status and timing.
- **MCP Support:** First-class support for Model Context Protocol, connecting your workflows to Cursor and Claude.

## 5. Built for Developers (Tech Stack)
- **Heading:** The Stack of the Future.
- **Description:** a8n is built with the latest technologies to ensure maximum performance and developer experience.
- **Stack Highlights:**
    - **Next.js 16 & React 19:** Modern app architecture.
    - **tRPC v11:** End-to-end type-safe API layer.
    - **Prisma & PostgreSQL:** Reliable data persistence.
    - **Better Auth:** Secure, multi-provider authentication.
    - **Tailwind CSS v4:** Sleek, performant styling.

## 6. How It Works
- **Step 1: Connect:** Set up triggers like manual starts, Stripe events, or Google Forms.
- **Step 2: Orchestrate:** Use the visual editor to link triggers to AI models, HTTP requests, or messaging platforms.
- **Step 3: Deploy:** Execute with confidence. a8n handles the reliability, scaling, and monitoring.

## 7. Pricing & Tiers
- **Free Tier:** Start building with essential nodes and execution history.
- **Pro Tier:** Unlock advanced AI nodes, higher execution limits, and premium support. Powered by Polar.sh.

## 8. Final CTA & Footer
- **Heading:** Ready to automate your world?
- **Subtext:** Join 200k+ developers building the next generation of automation.
- **Action:** [Start Building Now]
- **Footer Links:** Docs, GitHub, API Reference, Terms, Privacy.
